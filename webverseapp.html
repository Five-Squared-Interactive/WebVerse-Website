Thank you for trying WebVerse. The web application is a quick and easy way to get started in the Web Metaverse. A higher fidelity standalone version is available (with VR support). The web application works with the same worlds as the standalone version (with reduced performance).

What is WebVerse?

WebVerse is a reference implementation of the Web Metaverse concept, and for the protocols that will drive it. Put simply, the Web Metaverse uses the infrastructure and tools that already exist to exchange information about virtual worlds.
