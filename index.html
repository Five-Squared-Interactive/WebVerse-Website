Built on existing web technology

No need to reinvent the wheel. Take advantage of the thousands and thousands of tools already built for the web.

Web Metaverse is always available

The metaverse will be there whenever you'd like to go into it. It doesn't matter where you are or on what device (Disclaimer: Web Metaverse doesn't yet work on Mars).

Decentralized to the core

No one owns the Web Metaverse. As long as you can host a server, you can share your content with the rest of us. Being protocol-based, the Web Metaverse won't lock you into any one platform.

What is WebVerse?

WebVerse is a reference implementation of the Web Metaverse concept, and for the protocols that will drive it. Put simply, the Web Metaverse uses the infrastructure and tools that already exist to exchange information about virtual worlds.
