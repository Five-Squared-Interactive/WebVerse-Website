About WebVerse

WebVerse is a reference implementation of the Web Metaverse concept, and for the protocols that will drive it. Put simply, the Web Metaverse uses the infrastructure and tools that already exist to exchange information about virtual worlds.

The Web Metaverse consists of the following components:

  • World Servers: HTTP servers that distribute 3D models, scripts, markup documents, and anything else that a metaverse world is composed of.
  • Synchronization Servers: Servers that synchronize world entities across users in a metaverse world.
  • World Browsers: User-facing applications that render the world and allow the user to interface with it. WebVerse is an example of a World Browser.
