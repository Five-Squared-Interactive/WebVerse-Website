The Metaverse is already here! How will you shape it?

WebVerse is a Web Metaverse World Browser. The Web Metaverse allows developers to create and host immersive virtual worlds that can be accessed by users through world browsers. It offers a new way for people to engage with one another and with digital content.

How do I get started with WebVerse?

There is no need to purchase special equipment: WebVerse will run on billions and billions of devices - from smartphones to high-end gaming PCs - that already exist. Unlike other metaverse platforms, which require users to download and install software, WebVerse can be accessed through any modern web browser. This makes it easier for people to access virtual worlds and removes a significant barrier to entry.

Graphics rendering in a web browser will only get you so far, however. That is why we have created the WebVerse desktop client. The WebVerse client will take full advantage of your computer's graphical resources to immerse you even further in your worlds. It also supports SteamVR.

The Web Metaverse is designed to offer developers the utmost flexibility and independence. It is not a walled garden. The Virtual Environment Markup Language (VEML) and JavaScript APIs afford you complete control and allow you to customize every aspect of your environments. Even the foundation that WebVerse is built upon is fully open. With the Web Metaverse, you will always own what you build.

What is WebVerse?

WebVerse is a reference implementation of the Web Metaverse concept, and for the protocols that will drive it. Put simply, the Web Metaverse uses the infrastructure and tools that already exist to exchange information about virtual worlds.
